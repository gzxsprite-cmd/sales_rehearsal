# Sales Rehearsal AI — Pages Demo Repo

This repository is a document-driven prototype repo for **Sales Rehearsal AI**.

Purpose of this repo:
- define the business concept clearly
- define the v0.1 demo scope
- guide Codex to build a **GitHub Pages demo first**
- avoid unnecessary engineering complexity in the prototype stage

## Repo structure

Root markdown files:
- `00_PROJECT_POSITIONING.md`
- `01_V0.1_SCOPE_AND_GOAL.md`
- `02_USER_FLOW.md`
- `03_PAGE_AND_INTERACTION_SPEC.md`
- `04_AI_BEHAVIOR_AND_REVIEW_LOGIC.md`
- `05_PAGES_DEMO_RULES.md`
- `06_CODEX_EXECUTION_TASK.md`

Demo output directory:
- `docs/`

## Important rule

This repo is for a **Pages demo first**, not for a production app.

Codex must:
1. read all root-level markdown files first
2. keep root markdown files unchanged
3. generate the demo only inside `docs/`
4. prioritize direct Pages preview over engineering complexity

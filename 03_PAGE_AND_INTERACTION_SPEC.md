# 03 Page And Interaction Spec

## Overall interaction style

The demo must feel:
- lightweight
- clear
- fast to understand
- closer to a consumer-style experience than an enterprise system

Avoid:
- dense forms
- table-heavy screens
- dashboard feeling
- long explanatory text blocks

Prefer:
- one clear main action per screen
- simple cards
- chat-first interaction
- strong visual hierarchy
- obvious next step

## Page 1 — Home

### Goal
Get the user to start immediately.

### Must have
- product name / short description
- one main CTA: Start Rehearsal

### Optional
- a small recent practice block
- one short line explaining the value

### Should feel like
A launch page, not a system homepage.

---

## Page 2 — Setup

### Goal
Lock one specific conversation context quickly.

### Must have
- customer selection
- time selection (10 / 20 / 30)

### Should not have
- too many fields
- too much customization
- complex scenario builder behavior

### Should feel like
Fast context confirmation, not data entry.

---

## Page 3 — Chat

### Goal
Create the feeling of a real customer rehearsal.

### Must have
- clear chat layout
- customer speaks first
- user input area
- send action
- end action
- visible time concept
- visible context summary (light)

### Interaction requirement
The user should feel:
- pressure
- realism
- continuity

### Should not feel like
- a support chat
- a generic AI assistant
- a documentation search tool

---

## Page 4 — Review

### Goal
Turn one conversation into insight and next action.

### Must have
- outcome block (R/Y/G)
- good points
- unclear points
- risk points
- knowledge coaching
- suggested phrases
- next action CTA

### Review style
The review should feel like:
- a coach
- practical
- specific
- directly useful for the next round

### It should not feel like
- a generic AI summary
- a scoring dashboard
- a training report PDF

---

## General visual direction

- clean white/light background
- simple accent color system
- rounded cards
- soft shadows
- generous spacing
- minimal but readable text
- modern business-demo feel
